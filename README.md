
# QR code and RFID data logger to Database using ESP32CAM
## Requirements

- ESP32CAM
- RDM6300
- Arduino RFID Library for RDM6300
- Arduino ESP32QRCodeReader Library
- Arduino IDE
- Arduino Core for ESP32CAM Using Arduino Boards Manager
- Jumper Wires
- XAMPP


## Connection Map

| ESP32CAM | RDM6300       |
|----------| ------------- |
| D13      | Tx            |
| 5V       | 5V            |
| GND      | GND           |
